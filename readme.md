# Rake Tasks

## links
Install symbolic links for all the dotfiles.

> rake links[dir,force]
### Defaults
 * dir: $HOME
 * force: false

## vim
Create a .vim/ directory and download pathogen to .vim/autoload/.
> rake vim[dir,force]
### Defaults
 * dir: $HOME
 * force: false

## vim_plugins
Install the vim plugins to .vim/bundle/.
> rake vim_plugins[dir]
### Defaults
 * dir: $HOME/.vim/bundle/
